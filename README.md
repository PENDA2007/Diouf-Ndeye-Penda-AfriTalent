# AfriTalent
plateforme de mise en relation entre freelance tech et entreprises en Afrique.
# AUTEUR
** Diouf Ndeye Penda **
## Description
AfriTalent est une plateforme fictive qui permet aux freelances tech de trouver des missions et aux entreprises de recruter des talents. Ce projet a ete realise dans le
cadred'un projet semestriel.
## Technologies utilisees
-HTML5
-CSS
-Bootstrap 5
-JavaScript
-Git & GitHub

-Mode sombre/clair(locaStrorage)
